Walking Skeleton Service

Microservice Walking skeleton that is trying to integrate towards some free Soap WebServices (unfortunately they don't always work...) and exposing them via rest API. This project is work in progress application and there is a lot of TODOs on the way! Wanna help -> create pull request!
